## 📖 Overview

**Carbon Atom 3D Visualization** is an interactive educational web application that demonstrates the atomic structure of Carbon using modern web technologies.

Built with **Three.js** and **WebGL**, the project combines realistic 3D rendering with an intuitive user interface, allowing users to explore electron shells, inspect atomic properties, simulate ion formation, and learn important chemistry concepts in an engaging way.

Designed with a clean dark interface and smooth animations, the application delivers an immersive learning experience directly in the browser—no installation required.

---

## ✨ Features

### ⚛️ Interactive 3D Atom
- Real-time 3D Carbon atom visualization
- Smooth orbital animations
- Dynamic lighting and rendering
- Rotate and zoom the model freely

### 🧪 Electron Interaction
- Inspect individual electrons
- Interactive orbital exploration
- Educational information panel

### ⚡ Animation Controls
- Play animations smoothly
- Slow Motion mode
- Fast Forward mode

### ➕ Ion Simulation
- Add electrons
- Remove electrons
- Reset to the neutral Carbon atom
- Observe changes in atomic configuration

### 📚 Educational Content
Learn about:

- Atomic Structure
- Electron Configuration
- Electron Shells
- Valence Electrons
- Atomic Number
- Atomic Mass
- Isotopes
- Carbon Allotropes

### 🎨 Modern User Interface
- Minimalist dark theme
- Responsive design
- Smooth animations
- Easy-to-use controls
